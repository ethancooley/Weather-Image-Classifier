# Weather Image Classifier

Upload a weather photo and get an AI-powered classification with a GradCAM visual explanation.

**11 classes:** dew · fog/smog · frost · glaze · hail · lightning · rain · rainbow · rime · sandstorm · snow

Built with EfficientNetB3 fine-tuned on the [Weather Image Recognition dataset](https://www.kaggle.com/datasets/jehanbhathena/weather-dataset).
